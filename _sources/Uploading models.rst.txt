
Uploading models
^^^^^^^^^^^^^^^^^^^^


To upload user defined models to the server, in a web browser open the foloowing link http://localhost:5000. This should take you to the upload page and the followin page should open.

.. figure::  _static/upload.png
   :align:   center
   
   
Zip you OpenDSS model and click on "Choose files" to select the model. Next, click upload to upload the model to the server. One the upload is complete, you will automatically be taken to the main dashboard page. The new model should appear on the list of networks on the main page.
